# SQL Section

This section contains SQL solutions and explanations based on the assignment
requirements.

Although the assignment references Oracle and PL/SQL, the implementation is done
using MySQL, with clear mapping to Oracle concepts where applicable.

---

## Files

- `schema.sql`  
  SQL statements to create all required tables with constraints.

- `queries.sql`  
  SQL queries answering the assignment questions.

- `procedures.sql`  
  Stored procedures and functions implemented in MySQL.

- `sql_theory.md`  
  Theory answers covering SQL, DDL, DML, DCL, joins, RDBMS vs NoSQL.

---

## Topics Covered

- Database design and normalization
- Constraints and relationships
- Complex SQL queries and joins
- Stored procedures and functions
